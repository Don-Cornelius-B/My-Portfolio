# My-Portfolio

A personal portfolio website for **Don Cornelius Livi**, a Third-Year Computer Science and Engineering student at Sathyabama Institute of Science and Technology with a focus on **Cloud Computing and DevOps**.

## Features

- Responsive design with a Royal Marble & Gold theme
- About Me section highlighting DevOps/Cloud focus and academic background
- Skills section (Python, DevOps & CI/CD, Cloud Computing, Linux, and more)
- Projects section (ESP32 localized P2P communication system, Unity complex simulations)
- Contact section with GitHub profile link — no fake email addresses
- Accessibility: skip-to-content link and reduced-motion support

## Local Development

Open `index.html` directly in your browser — no build step required.

## Deployment

This site is designed for [GitHub Pages](https://pages.github.com/).

To enable:
1. Go to **Settings → Pages** in this repository.
2. Set the source to the `main` branch, root folder.
3. GitHub Pages will serve `index.html` automatically.

Live site (after enabling GitHub Pages): `https://don-cornelius-b.github.io/My-Portfolio/`
